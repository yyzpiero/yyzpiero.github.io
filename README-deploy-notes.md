# Deploy Notes

Copy the contents of this `github-ready` directory into the root of `yyzpiero.github.io`.

Editable homepage content lives in:

- `_data/home.yml`: hero text, links, profile subtitle, paper count
- `_includes/home/about.md`: about section
- `_includes/home/blog.md`: blog placeholder text when no posts exist
- `_data/news.yml`: short homepage news items
- `_data/featured_publications.yml`: selected publication cards shown on the homepage
- `_data/home_publications.yml`: full publication cards shown on the publications page
- `_data/projects.yml`: project cards
- `projects/`: long-form project detail pages
- `_data/profile.yml`: profile image, experience, interests, and social links
- `_bibliography/publications.bib`: BibTeX source for the publications page
- `_posts/`: future blog posts
- `cv.md`: full CV page

GitHub Pages will rebuild `https://yyzpiero.github.io/` after the changes are pushed to `master`.
